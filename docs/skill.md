# Pixel Defender — Skills y tecnologías

## Lenguajes
- **Dart** 3.12+ — Lenguaje principal
- **Kotlin** — Plataforma Android nativa
- **Java** — Gradle / Android SDK

## Frameworks y motores
- **Flutter** 3.44 — Framework UI multi-plataforma
- **Flame** 1.18 — Motor de juego 2D para Flutter
- **flame_audio** 2.10 — Audio (música + SFX)
- **shared_preferences** 2.3 — Persistencia local

## Patrones de diseño
- **Object Pooling** — Reciclaje de entidades (balas, enemigos, pickups)
- **Component-Entity** — Arquitectura de componentes Flame
- **Catálogo/config** — Data-driven design (enemigos, armas, mejoras)
- **Singleton** — Managers (audio, juego, guardado)
- **Observer** — Eventos del GameManager → logros

## Herramientas
| Herramienta       | Uso                                    |
|-------------------|----------------------------------------|
| Android Studio    | SDK Manager, emulador, profiling       |
| VSCode            | Editor con extensión Flutter/Dart      |
| Gradle            | Build system Android                   |
| ADB               | Debug y deploy en dispositivo          |
| GitHub            | Control de versiones y CI/CD           |

## Habilidades requeridas
- Desarrollo de videojuegos 2D con Flame
- Programación Dart async (audio, persistencia)
- Optimización de renders (Canvas vs Widgets)
- Sistemas de pooling para rendimiento en móvil
- Diseño de sistemas RPG (progresión, upgrades, logros)
